## How to Use

### Local Development

1. Open `index.html` in any web browser
2. Navigate through the blog posts using the cards or navigation links
3. No server required - it's all static HTML/CSS

### GitHub Pages Deployment

1. Create a new GitHub repository
2. Upload all files to the repository
3. Go to Settings > Pages
4. Select the main branch as the source
5. Your site will be live at `https://yourusername.github.io/repository-name/`

### Customization

- **Colors**: Edit the gradient colors in `css/style.css` (search for `#667eea` and `#764ba2`)
- **Content**: Modify any HTML file in the `posts/` directory
- **Layout**: Adjust the grid layout in `css/style.css`
